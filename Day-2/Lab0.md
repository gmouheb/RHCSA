# Lab 0: User and Group Management

This lab focuses on setting up a shared group environment with specific user configurations and password policies.

## Requirements

Set up a shared group environment that meets the following requirements:

1. Create two groups: sales and account.

2. Create users joana, john, laura, and beatrix. Make sure they have their primary group set to a private group that has the name of the user.

3. Make joanna and john members of the group sales, and laura and beatrix members of the group account.

4. Set a password policy that requires users to change their password every 90 days.
